### CSS란?

HTML 등의 마크업 언어로 작성된 문서가 실제로 웹사이트에 표현되는 방법을 정하는 스타일시트 언어

### 양식

`head`태그 안의 `style` 태그에 작성

```markdown
<head>
    <meta charset="UTF-8">
    <style>
        h1 {
            background-color : red;
        }
        h2 {
            background-color : green;
        }
    </style>
</head>
```
